### The Vision
**Convertit.space** is a digital laboratory dedicated to transforming raw data, ideas, and static concepts into dynamic spatial realities. 

### Project Structure
* `index.html` - The core landing page (Optimized for speed).
* `/assets` - Contains renamed, high-resolution imagery (formerly "Replica" files).
* `/docs` - Project documentation and vision statements.

### How to Run Locally
1. Clone the repository.
2. Open `index.html` in any modern web browser.

### Tech Stack
* **HTML5 / CSS3**
* **Tailwind CSS** (via CDN for rapid deployment)
* **Google Fonts** (Plus Jakarta Sans)
